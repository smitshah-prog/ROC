# Security Policies

## 🔑 Authentication

* Username + Password login
* JWT-based sessions

---

## 🔒 Password Policy

* Minimum 8 characters
* Strong password recommended

---

## 🛡️ LAN Security

* Restrict access to internal network only
* Use firewall rules
* Disable external access

---

## 📁 Data Protection

* Store data locally
* Restrict folder access
* Role-based permissions

---

## 🧾 Audit Logs

* Track all user activity
* Store logs securely

---

## 🚨 Risk Considerations

* Server machine failure
* Unauthorized LAN access

---

## 💾 Backup Policy

* Daily backups required
* External storage recommended

---

## 🔐 Future Enhancements

* HTTPS within LAN
* 2FA authentication
